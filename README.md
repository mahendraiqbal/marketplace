# Marketplace

## Features

- Login
- get product
- get one product
- add cart

## Installation

1. Clone this repository
```bash
  $ https://github.com/mahendraiqbal/marketplace
```

2. Download all packages
```bash
  $ npm install
```

4. Run npm start

```bash
  $ npm start
```

## Endpoint

```bash
  / => Login
  /product => get all product
  /product:id => get one product
  /carts/user:id => get cart product per user
```
